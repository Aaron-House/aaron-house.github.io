# AI가 답변을 만드는 법 ## 1. 질문 입력 사용자가 자연어로 질문한다. "OO 브랜드는 어떤 회사야?" 같은 방식이다. ## 2. 정보 수집 AI는 공개된 웹사이트, 뉴스, 리뷰, 백과사전, 학술자료 등을 탐색한다. 신뢰할 수 있는 출처일수록 답변에 포함될 확률이 높다. ## 3. 맥락 분석 (Grounding) 단순히 문장을 수집하는 게 아니라, 질문 의도에 맞게 맥락을 재구성한다. 최근 구글 내부 문서 유출에서도, AI는 **SiteFocus Score, Host NSR** 등 다양한 지표를 통해 품질과 맥락을 평가한다고 확인되었다. ## 4. 답변 생성 최종적으로 AI는 문장을 생성한다. 이때 NavBoost 같은 시스템을 통해 "사용자가 만족할 답변"을 보정한다. 클릭 후 이탈이 적고, 만족도가 높은 콘텐츠가 장기적으로 더 많이 반영된다. ## 5. 브랜드가 답변에 포함되는 조건 - 신뢰할 수 있는 출처인지 - 토픽 권위가 있는지 - 최근 업데이트가 이루어졌는지 - 외부 인용(백링크, 언론보도 등)이 있는지 ## 6. 결론 AI 답변은 단순 요약이 아니다. GEO의 목표는 "브랜드가 이 생성 과정 속에 자연스럽게 포함되도록 설계하는 것"이다.
