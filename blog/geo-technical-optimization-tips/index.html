# GEO를 위한 테크니컬 팁 ## 1. 크롤링/색인 관리 - robots.txt 최적화: 불필요한 페이지 차단 - XML 사이트맵 제출: 전체 구조 전달 - 캐노니컬 태그: 중복 방지 ## 2. 구조화 데이터 Schema.org 기반 JSON-LD 적용은 필수다. FAQ, Product, Article 등 다양한 마크업을 통해 AI가 콘텐츠 의미를 정확히 이해하도록 돕는다. ## 3. Core Web Vitals - LCP: 주요 콘텐츠 로딩 속도 - INP: 사용자 입력 반응 속도 - CLS: 시각적 안정성 3초 이상 걸리는 사이트는 GEO 경쟁에서 밀릴 수 있다. ## 4. 보안 & 모바일 대응 - HTTPS 적용: 신뢰 확보 - 반응형 웹 디자인: 모바일 색인 대비 - SPA는 SSR/프리렌더링으로 보완 필요 ## 5. 실행 체크리스트 - GSC 등록 여부 확인 - 사이트맵 최신화 - 스키마 마크업 테스트 - CWV 측정 및 개선 - 정기 기술 감사 진행 ## 6. 결론 GEO는 콘텐츠 전략 못지않게 기술적 기반이 중요하다. **"콘텐츠는 메시지, 기술은 토대"** 두 가지가 함께 작동할 때 AI가 신뢰하는 답변 출처가 될 수 있다.
